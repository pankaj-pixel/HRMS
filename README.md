# HRMS Lite (Human Resource Management System)

HRMS Lite is a lightweight full-stack web application designed to manage basic employee operations such as employee registration, listing, attendance, and department management.  
The application is built with a modern frontend and a RESTful backend and is fully deployed on cloud platforms.
